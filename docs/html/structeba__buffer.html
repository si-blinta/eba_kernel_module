Detailed Description

struct eba_buffer - Tracking structure for each allocated memory chunk. @lock: Spinlock to protect this buffer's data. @type: Buffer type. @address: Address of the allocated memory block. @size: Requested allocation size in bytes. @expires: Time when the allocation expires. @node: List node to chain all allocations.

This structure tracks each memory allocation made from the EBA memory pool.
