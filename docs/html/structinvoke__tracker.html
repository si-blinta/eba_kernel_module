Detailed Description

struct invoke_tracker - Tracks the status of an EBA protocol invocation. @iid: Unique identifier for the invocation. @pid: Process identifier associated with the invocation. @done: Boolean flag indicating whether the invocation is complete. @status: Current status of the invocation (see INVOKE_STATUS).
