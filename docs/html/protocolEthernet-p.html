<Ethernet> Protocol Reference

Detailed Description

send_raw_ethernet_packet - Send a raw Ethernet packet. @payload: Pointer to the payload data to include in the packet. @payload_len: Length of the payload in bytes. @dst_mac: Pointer to the destination MAC address (typically a 6-byte array). (e.g., ETH_P_IP, ETH_P_ARP) in host byte order. @ifname: Network interface name (e.g., "eth0") through which the packet is transmitted.

This function allocates an sk_buff and constructs the Ethernet header by setting the destination and source MAC addresses as well as the specified protocol. It then appends the provided payload and transmits the packet using the appropriate network device.

Return: 0 on success or a negative error code on failure.
