Detailed Description

struct ebp_op_alloc_args - Arguments for the exposed EBA "alloc" operation. @size: Number of bytes to allocate. @life_time: Lifetime for the allocation (in a defined time unit). @buffer_id: Field that the target node will fill with the allocated buffer's unique identifier.

These parameters are provided by a remote node to request a buffer allocation.
